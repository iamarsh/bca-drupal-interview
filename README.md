# Drupal Principal Developer Interview Repository

This repository contains sample Drupal modules designed for a Principal Developer technical interview.

**IMPORTANT NOTICE**: This repository contains only sample code for interview purposes and does not represent any real project or organization. No proprietary or sensitive code is included.

## Repository Structure

```
bca-drupal-interview/
├── README.md
└── web/
    └── modules/
        └── custom/
            ├── bca_reports/     # Code review exercise
            └── bca_task/        # Pseudo-coding exercise
```

## Modules Overview

### 1. bca_reports - Code Review Module
A report listing and management module for code review and discussion.

**Interview Task**: Review the implementation and discuss:
- Code architecture and patterns
- Performance considerations
- Drupal best practices
- Potential improvements

### 2. bca_task - Pseudo-Coding Exercise
A service class with method signatures and pseudo-code for verbal implementation.

**Interview Task**: Walk through implementation approach, discussing:
- API integration patterns
- Caching strategies
- Error handling and logging
- Data validation

## Getting Started

```bash
git clone https://github.com/iamarsh/bca-drupal-interview.git
cd bca-drupal-interview
```

This repository is designed for code reading and discussion. No Drupal installation required.
